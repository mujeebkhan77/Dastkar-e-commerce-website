# 🛒 Dastkaar Ecommerce Website

A fully functional frontend ecommerce website built using **HTML, CSS, and JavaScript**.  
This project simulates a real-world online store with product management, cart system, and admin functionality.


## 🌐 Live Demo

👉 https://dastkar-e-commerce-website.vercel.app/


## 📸 Project Overview

This project is a frontend-based ecommerce system that allows users to browse products, view product details, add items to cart, and simulate a basic checkout flow. It also includes an admin panel for managing products dynamically.


## ✨ Features

### 🛍️ User Side
- Product listing page (Home / Shop)
- Product detail page
- Add to cart functionality
- Cart page with quantity management
- Order summary page
- Responsive and modern UI

### ⚙️ Admin Panel
- Add new products dynamically
- Upload product images
- Manage product details (name, price, description, category)
- Products reflect instantly on website

### 🧠 System Features
- Dynamic product rendering
- Persistent cart system
- Clean UI with ecommerce theme
- Page navigation between shop, product, cart, and admin


## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Local Storage (for data handling)

1. Clone the repository:
```bash

````

2. Open the project folder

3. Run `index.html` in your browser

No installation required.
